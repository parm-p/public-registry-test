MCP Registry - Public Registry Test

Welcome to the public MCP (Model Context Protocol) registry test site. This registry provides a list of available MCP servers following the official MCP registry specification.

API Endpoints

GET /v0.1/servers

Lists all available MCP servers in this registry.

Available Servers

GitHub MCP Server

Description: GitHub's official MCP Server - connects AI tools directly to GitHub's platform for repository management, issue & PR automation, CI/CD intelligence, code analysis, and team collaboration.

Repository: github/github-mcp-server
